Introduction

Ce module ajoute un système de filtres produits avancés dans Odoo. Vous pouvez définir des filtres (ex. Couleurs, Matériaux, Tailles) et leurs valeurs (ex. Rouge, Vert · Bois, Métal · S, M, L) directement sur la fiche produit, puis permettre aux clients de filtrer dans la boutique du site web.

✅ Expérience d'achat améliorée : navigation plus rapide, conversions optimisées.

Installation

  1. Connectez‑vous en tant qu'Administrateur sur votre instance Odoo.
  2. Allez dans Applications → recherchez « Filtres produits avancés ».
  3. Cliquez sur Installer et patientez.

Une fois installé, le module est actif par défaut. Si vous utilisez plusieurs sites, vérifiez la disponibilité sur le site concerné.

Créer des filtres sur un produit

Deux méthodes complémentaires sont disponibles :

1️⃣ Depuis la fiche produit

  1. Ventes → Produits → sélectionnez un produit.
  2. Dans l'onglet Filtres, ajoutez les filtres et leurs valeurs (ex. Couleurs : Rouge, Vert, Bleu).
  3. Enregistrez la fiche produit.

2️⃣ Depuis le menu Configuration → Filtres

Vous pouvez aussi créer les filtres globalement : Ventes → Produits → Configuration → Filtres.

  1. Cliquez sur Créer pour ajouter un filtre (ex. Couleurs).
  2. Choisissez un Type : Radio (valeurs texte) ou Color (palette de couleurs pour chaque valeur).
  3. Enregistrez et associez ce filtre aux produits souhaités.
Remarque : les filtres Color affichent des pastilles de couleur côté boutique.

Utilisation sur la Boutique (site web)

  1. Site Web → Boutique.
  2. La colonne latérale affiche la liste des filtres disponibles.
  3. Cliquez sur une ou plusieurs valeurs pour filtrer les produits.

Recherche & filtrage en back‑office

Réutilisez ces filtres pour accélérer la recherche interne (recherches favorites, domaines personnalisés).

Astuces & bonnes pratiques

  • Nommage clair pour éviter les doublons.
  • Valeurs cohérentes (ex. « Bleu » vs « bleu »).
  • Nettoyage : supprimez les filtres inutilisés.

FAQ & Dépannage

Les filtres n'apparaissent pas

  • Vérifiez que le produit est publié.
  • Assurez au moins un filtre avec une valeur.
  • Rafraîchissez la page ou videz le cache.
